# RSS Aggregator

A web application for reading multiple RSS feeds in one place. Allows adding RSS channels, viewing posts, and reading their content directly in the app.

**Features:** URL validation, i18n (Russian), auto-update every 5 seconds, responsive design.

**Tech Stack:** Pure JavaScript, Bootstrap 5, i18next, Axios, Vite, yup validation.

## Quick Start

```bash
git clone https://github.com/Freemason-EAG/frontend-project-11.git
cd frontend-project-11
npm install
npm run dev
```
